# Collective Memory Skill

This folder is the installable skill package for the Collective Memory workflow.

## Install from GitHub

Use this folder URL with the Codex skill installer or the equivalent GitHub import flow:

`https://github.com/nestorfernando3/collective-memory-ui/tree/main/collective-memory-skill`

## What it does

- Runs `/memoria` workflows against the memory database.
- Keeps the graph balanced with `strong`, `exploratory`, and `discarded` connections.
- Uses `research_sync.js` as the connection engine entrypoint.
